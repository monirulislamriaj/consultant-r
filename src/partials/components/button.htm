Standard Buttons

The default button style for Consultant using the .btn class. Customize the size of the buttons using .btn-lg and .btn-sm

Button default
Button Large
Button Small

                      <a class="btn btn-primary">Button default</a>
                      
                      <!--Default Large Button-->
                      <a class="btn btn-primary btn-lg">Button Large</a>
      
                      <!--Default Small Button-->
                      <a class="btn btn-primary btn-sm">Button Small</a>
